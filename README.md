# Thermora: Journey to Mars

Thermora: Journey to Mars is a 2D arcade space shooter developed in Unity as part of the LUT University Game Development course.

## Game Overview

Take control of the spacecraft **Thermora** on a challenging mission to Mars. Navigate through enemy waves, collect power-ups, upgrade your weapons, and manage your ship's heat levels to prevent overheating during intense combat. Survive increasingly difficult levels and achieve the highest score possible.

## Features

* Progressive enemy waves with increasing difficulty
* Weapon overheating and heat management system
* Power-ups and weapon upgrade mechanics
* Score tracking system
* Lives and shield systems
* Level progression
* Main menu and game over screens
* Sound effects and background music
* Victory and defeat game states

## Controls

| Key             | Action       |
| --------------- | ------------ |
| A / Left Arrow  | Move Left    |
| D / Right Arrow | Move Right   |
| Space           | Shoot        |
| R               | Restart Game |
