# Thermora: Journey to Mars

Thermora: Journey to Mars is a 2D arcade space shooter developed in Unity for the LUT Game Development course.

## Game Overview

Players control the spacecraft Thermora on a mission to Mars while defending against waves of enemies and hazards. The game features weapon upgrades, power-ups, shields, and a heat management system that requires players to avoid overheating during combat.

## Features

* Enemy waves with increasing difficulty
* Weapon overheating system
* Power-ups and weapon upgrades
* Score, lives, shield, and level systems
* Main menu and game over screens
* Sound effects and background music

## Controls

* A / Left Arrow – Move Left
* D / Right Arrow – Move Right
* Space – Shoot
* R – Restart

## Repository Contents

* Scripts.zip – C# source code
* Scenes.zip – Unity scenes
* Sprites.zip – Graphics and visual assets
* Prefabs.zip – Unity prefabs
* Audio files – Sound effects and music
